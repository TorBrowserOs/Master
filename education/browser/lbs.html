Got older web apps?

Legacy Browser Support can help

Your school may use web apps built for browsers that are now a few versions old. Deploying Chrome unlocks the benefits of the modern web, offering more security for IT plus speed and the ability to run the latest educational apps for students and teachers.

Chrome’s Legacy Browser Support allows users to switch automatically between Chrome and another browser. IT specifies which sites should launch into a second browser and deploys this Chrome policy for the school organization.
